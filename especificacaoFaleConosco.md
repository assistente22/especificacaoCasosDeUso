# Especificação de caso de uso 

## FALE CONOSCO 

##### Objetivo:
Usuário conseguir informações de contato da organização e/ou divulgar uma vaga

##### Requisitos: 
Clicar em 'Fale conosco' 

##### Atores:
Usuários (aluno, professor orientador, gerente de estágio) e visitantes 

##### Pré-condições: 
_Nenhuma_ 

##### Pós-condições: 
Divulga uma vaga de estágio pelo sistema ou retorna para a tela inicial 

##### Fluxo principal: 
1. O sistema apresenta a tela inicial ao usuário; 
2. O usuário clica em 'Fale conosco' ; 
3. O usuário lê as informações de contato ; 
4. O usuário clica em retornar ; 
5. O usuário retorna para a tela inicial. 

##### Fluxo alternativo:  Usuário quer divulgar uma vaga de estágio

3. 1. O usuário clica em 'Divulgar uma vaga de estágio' ; 
    2. O usuário vai para a tela de divulgar vaga ; 
    3. O usuário insere as informações da vaga; 
    4. O usuário envia as informações ao sistema; 
    5. O usuário permanece na mesma tela.
